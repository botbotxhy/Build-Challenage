# AS2 - Sales Data Analysis (Functional / Stream-Style Python)

## Overview
This project that analyzes sales data using functional programming techniques. Built as a learning exercise to practice `map`, `filter`, `reduce`, and `groupby` instead of traditional loops.


## Dataset
A synthetic dataset `sample_sales.csv` is included.
And the `sample_sales.csv` is generated by AI

### Columns
- order_id
- date (YYYY-MM-DD)
- region
- salesperson
- product
- quantity (int)
- unit_price (float)


## Run Analysis
```
python main.py
```
## Test
```
python -m unittest test_sales_analysis.py
```
